# Windows 11 Enterprise Privacy Toolkit

> Enterprise-grade privacy toolkit for Windows 11.

## Goals

This project provides a modular PowerShell toolkit to improve privacy on Windows 11 Enterprise while keeping:

- ✅ Windows Update
- ✅ Microsoft Account
- ✅ Microsoft Store
- ✅ winget
- ✅ Microsoft Defender
- ✅ Windows Activation

fully functional.
